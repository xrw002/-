国家代码通常指的是ISO 3166-1 alpha-2标准下的国家代码，这是由国际标准化组织(ISO)定义的两个字母的代码，用于表示世界上的各个国家或地区。以下是一些常见的国家代码示例：
````
- 中国：    CN
- 美国：    US
- 英国：    GB
- 法国：    FR
- 德国：    DE
- 日本：    JP
- 澳大利亚：AU
- 加拿大：  CA
- 印度：    IN
- 巴西：    BR
- 俄罗斯：  RU
````

这些只是一小部分示例，实际上ISO 3166-1 alpha-2标准包含了世界上所有国家和地区的代码。如果你需要完整的列表，可以访问ISO官方网站或使用相关的数据库查询。
